📖 About IRID Institute

IRID Institute (Institute for Research, Insights and Dialogue) is an independent research and policy institution dedicated to advancing evidence-based knowledge, generating strategic insights, and fostering structured dialogue across sectors.

The Institute operates as a platform for rigorous research, informed public discourse, and collaborative engagement between academia, policymakers, industry leaders, and civil society.

IRID Institute is committed to strengthening institutions, promoting analytical thinking, and contributing to sustainable and inclusive development.

🎯 Mission

To produce credible research, deliver actionable insights, and facilitate constructive dialogue that informs policy, strengthens institutions, and empowers future leaders.

🌍 Vision

To become a leading regional center for research excellence and policy dialogue, shaping informed decision-making and long-term development strategies.

🛠 Development Setup

To run the project locally:

Clone the Repository
git clone <YOUR_GIT_URL>
Navigate to Project Directory
cd irid-institute
Install Dependencies
npm install
Start Development Server
npm run dev

The application runs with hot-reload enabled for efficient development.
